## git-bug bug title edit

Edit a title of a bug

```
git-bug bug title edit [BUG_ID] [flags]
```

### Options

```
  -t, --title string      Provide a title to describe the issue
      --non-interactive   Do not ask for user input
  -h, --help              help for edit
```
